## 简介 Introduction

Kumo O Tagayasu 耕云钓月是一个注重田园体验的综合整合包，补全了大部分材料的量产，旨在给予玩家一个休闲惬意的游戏体验。

## 如何使用 How to use

下载 Release 的文件后拖入启动器中安装。

要获得服务端，或者自定义游戏体验，请参考 [服务端搭建指南](./Server.md)。
