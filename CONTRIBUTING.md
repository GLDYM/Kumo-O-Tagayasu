# 开发 & 贡献 Development & Contribution

项目下有两个分项目：

There are two sub-projects under this project:

- Kumo O Tagayasu 耕云钓月整合包
- Mandalas GUI - Dark Mode KOTS Compat：适配织彩钓月与耕云钓月整合包的 Mandalas GUI - Dark Mode 资源包拓展

- Kumo O Tagayasu
- Mandalas GUI - Dark Mode KOTS Compat

如果想要对本项目进行修改/PR，可以 Fork 本项目后 Clone 到本地进行修改。

If you want to modify this project, you can fork this project and clone it to your local machine.

>  该整合包使用数据包驱动修改，这些配方位于 `.pakku/overrides/config/openloader/data/kots_datapack/data`中。
> 
> This mod uses data pack driven mods, the recipes are located in `.pakku/overrides/config/openloader/data/kots_datapack/data`.

项目接受如下 PR：

This project accepts the following PRs:

- FTBQ 任务；
- Tag 缺失；
- 对于模组的增删修改，并说明增删修改的理由；
- 对于整合包配方的修改，并说明修改的理由；
- 对于整合包配置文件的修改，并说明修改的理由；
- 对整合包内置资源包的补充。

- FTBQuests;
- Tag missing;
- Mod addition, deletion, modification, with the reason for the modification;
- Modification of recipes, with the reason for the modification;
- Modification of the configuration file, with the reason for the modification;
- Additions to the built-in resource pack of the mod;
